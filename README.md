地理院タイルからボクセル状の地形を生成する実験用プログラム。

レンダリングが非常に高コストで、良い方法とは言えません。

> [!CAUTION]
> ブラウザ側でGBオーダーの非常に多くのメモリを使用します。
> メモリに余裕がない場合は、このプログラムを実行しないでください。
